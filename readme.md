<h1 align="center">Send Email</h1>
<p>Micro serviço que envia e-mails apartir de uma fila RabbitMQ</p>
<p>Apartir dos dados consumidos da fila envia o e-mail para o destinatário</p>
<pre>
    {
        "to":"user@gmail.com", 
        "subject":"Example test", 
        "text":"This a example test"
    }
</pre>

<h2 align="center">Tecnologias</h2>
<div align="center">
    Spring Boot | RabbitMQ | Kotlin 1.9 | Java 17 | Gradle
</div>
